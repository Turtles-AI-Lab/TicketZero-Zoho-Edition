TicketZero AI - Local Testing Guide

Quick Start (5 Minutes)

Step 1: Access Zoho Desk Developer Space

  1. Login to your Zoho Desk instance
  2. Navigate to Setup → Developer Space → Widgets
  3. Click "Create New Widget"

Step 2: Upload Widget Files

  1. Widget Name: TicketZero AI
  2. Upload plugin-manifest.json from 02_APP_FILES/
  3. Upload widget files:
    • app/ticket-detail.html - Main widget UI
    • app/background.html - Background service
    • icon.png - Widget icon
  4. Set widget location: Ticket Detail Page - Right Sidebar

Step 3: Configure API Endpoints

For local testing, configure these endpoints:

{
    "api_base": "http://localhost:5000",
    "endpoints": {
        "analyze": "/api/analyze",
        "resolve": "/api/resolve",
        "statistics": "/api/statistics"
    }
}

Step 4: Run Local API Server (Optional)

To test with full functionality:

cd 02_APP_FILES/api
python -m pip install flask flask-cors
python zoho_integration.py

Server will run on http://localhost:5000

Testing Without Backend

The widget includes mock data for UI testing without a backend:

  • Open any ticket in Zoho Desk
  • Widget will display in right sidebar
  • Click "Analyze" to see mock AI analysis
  • Review suggested resolutions

Troubleshooting

Widget Not Appearing?

  • Check widget is enabled in Developer Space
  • Verify user has permission to view widgets
  • Clear browser cache and reload

API Connection Failed?

  • Widget works in demo mode without backend
  • For full functionality, ensure local server is running
  • Check browser console for error messages
