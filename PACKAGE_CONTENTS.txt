INTEGRATION OVERVIEW:
---------------------

This package integrates THREE key technologies:

1. ZOHO DESK (Primary)
   - Hosts the widget in ticket detail page
   - Manages ticket workflow

2. ZOHO ASSIST (Secondary)
   - Provides remote machine access
   - Executes local operations (disk cleanup, service restarts)

3. MICROSOFT GRAPH API (External)
   - Manages Azure AD operations
   - Handles password resets, account unlocks, licenses

4. LOCAL LLM (AI Engine)
   - Analyzes tickets
   - Classifies issues
   - Routes to appropriate API

WORKFLOW:
---------

Ticket Created → LLM Analyzes → Routes to API → Executes Fix → Updates Ticket
