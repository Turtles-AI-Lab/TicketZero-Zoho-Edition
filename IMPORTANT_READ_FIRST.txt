3. WHAT EACH COMPONENT HANDLES
==================================================================

LOCAL LLM (AI ENGINE):
- Reads and analyzes ticket text
- Classifies issue (password_reset, disk_full, etc.)
- Determines confidence (0-100%)
- Chooses appropriate API

MICROSOFT GRAPH API handles:
- Password resets
- Account unlocks
- License assignments
- Group management
- All Azure AD operations

ZOHO ASSIST API handles:
- Disk cleanup
- Service restarts
- Software installation
- Printer fixes
- All local machine operations

ZOHO DESK API handles:
- Ticket updates
- Status changes
- Customer notifications
- Escalations
