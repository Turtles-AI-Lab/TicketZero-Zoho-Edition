2. Step-by-Step Integration Guide

Architecture Overview:

Zoho Desk (Ticket) → AI Analysis → Microsoft Graph (Azure AD) + Zoho Assist (Remote Access) → Resolution

Installation & Configuration Steps:

Step 1: Zoho Desk Widget Setup

  1. Navigate to Zoho Desk → Setup → Developer Space → Widgets
  2. Upload plugin-manifest.json from our package
  3. Configure widget to appear in ticket detail page
  4. Grant API permissions (Tickets, Contacts, Departments)

Step 2: Microsoft Graph Configuration

  1. Register application in Azure AD portal
  2. Grant required Graph API permissions:
    • User.ReadWrite.All
    • UserAuthenticationMethod.ReadWrite.All
    • Directory.ReadWrite.All
  3. Generate client credentials
  4. Configure in widget settings

Step 3: Zoho Assist Setup

  1. Enable API access in Zoho Assist
  2. Deploy unattended agent to end-user machines
  3. Configure automation technician account
  4. Link with Zoho Desk widget

Step 4: Operational Workflow

  1. Ticket Creation: User submits ticket in Zoho Desk
  2. AI Analysis: Widget analyzes ticket content
  3. Classification: Determines issue type and confidence
  4. Automated Resolution:
    • For Azure AD operations → Microsoft Graph API
    • For local machine operations → Zoho Assist API
  5. Update Ticket: Status changed to resolved with notes
